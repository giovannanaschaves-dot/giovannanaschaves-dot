# Página Metalinguística sobre UX/UI e Heurísticas de Nielsen (HTML)

Projeto desenvolvido em HTML com o objetivo de aplicar conceitos de estruturação de páginas web, além de abordar de forma metalinguística temas como UX/UI e as 10 heurísticas de Nielsen.

## 🚀 Tecnologias

* HTML

## 📚 O que aprendi

* Estruturação de páginas com HTML semântico
* Organização de conteúdo em seções (header, main, section, footer)
* Estilização básica utilizando CSS
* Aplicação de conceitos de UX/UI
* Compreensão das heurísticas de Nielsen
* Criação de conteúdo explicativo dentro de uma interface web
* Uso de boas práticas de organização visual

## ▶️ Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

## 📌 Observações

Este projeto foi desenvolvido como atividade acadêmica com o objetivo de praticar HTML de forma metalinguística, explicando conceitos importantes de design de interfaces dentro da própria página. Também foram utilizados recursos de inteligência artificial como apoio para organização das ideias e estruturação do conteúdo.
