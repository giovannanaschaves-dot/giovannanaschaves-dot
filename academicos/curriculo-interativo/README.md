# Currículo Interativo (HTML & CSS)

Projeto de front-end desenvolvido com HTML e CSS com o objetivo de criar um currículo digital interativo, estruturado para navegação e apresentação de informações de forma clara e visual.

## 🚀 Tecnologias

* HTML
* CSS

## 📚 O que aprendi

* Estruturação de páginas utilizando HTML semântico
* Estilização e organização visual com CSS
* Criação de layout para apresentação de informações
* Organização e separação de conteúdo e estilo
* Boas práticas de estruturação de páginas web

## ▶️ Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

## 📌 Observações

Este projeto foi desenvolvido como prática de front-end e tem como foco a criação de um currículo interativo, permitindo apresentar informações profissionais de forma mais dinâmica do que um currículo tradicional.
