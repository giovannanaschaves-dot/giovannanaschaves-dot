Exercícios de Programação Orientada a Objetos em Java
Este repositório contém exercícios desenvolvidos em Java com foco na prática de lógica de programação e conceitos iniciais de Programação Orientada a Objetos.

🚀 Tecnologias
Java

📚 O que foi praticado
Estruturas de repetição (for, while)
Condicionais (if/else, switch)
Manipulação de entrada do usuário
Operadores matemáticos e lógicos
Lógica de programação aplicada em Java
Encapsulamento
Criação de classes e objetos
Construtores
Getters e Setters
Validação de dados em métodos

📁 Exercícios

🔹 Tabuada Inteligente
Geração de tabuada de 1 a 10
Validação de entrada do usuário
Identificação de números pares e ímpares

🔹 Simulador de Caixa Eletrônico
Menu interativo com múltiplas opções
Controle de fluxo com laço de repetição
Operações de depósito e saque
Validação de saldo e entradas inválidas

🔹 Gerenciador de Produto (Estoque Simples)
Criação de classe com atributos privados (encapsulamento)
Implementação de construtor para inicialização do objeto
Uso de getters e setters para controle de acesso aos dados
Criação de métodos de comportamento (adicionarEstoque, vender, exibirResumo)
Validação de entradas inválidas e controle de estoque
Simulação de operações reais de estoque com testes positivos, negativos e casos extremos

▶️ Como executar
Acesse a pasta do exercício desejado
Compile o arquivo:
javac Main.java
Execute:
java Main

📌 Observações
Os exercícios foram desenvolvidos durante a disciplina de Programação Orientada a Objetos, com foco na consolidação da lógica de programação em Java.
