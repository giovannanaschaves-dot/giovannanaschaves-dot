# Exercícios de Programação Orientada a Objetos (Java)

Exercícios desenvolvidos em Java com o objetivo de praticar lógica de programação e conceitos iniciais de Programação Orientada a Objetos, aplicando validações, controle de fluxo e manipulação de dados.

## 🚀 Tecnologias

* Java

## 📚 O que aprendi

* Estruturas de repetição (for, while)
* Condicionais (if/else, switch)
* Manipulação de entrada e saída de dados
* Operadores matemáticos e lógicos
* Criação de classes e objetos
* Uso de atributos privados (encapsulamento)
* Implementação de construtores
* Criação de getters e setters
* Desenvolvimento de métodos de comportamento
* Validação de dados e regras de negócio

## 📁 Exercícios

### 🔹 Tabuada Inteligente

* Geração de tabuada de 1 a 10
* Validação de entrada do usuário
* Identificação de números pares e ímpares

### 🔹 Simulador de Caixa Eletrônico

* Menu interativo com múltiplas opções
* Controle de fluxo com laço de repetição
* Operações de depósito e saque
* Validação de saldo e entradas inválidas

### 🔹 Gerenciador de Produto (Estoque Simples)

* Criação de classe representando um produto
* Uso de atributos privados para controle de acesso
* Implementação de construtor para inicialização dos dados
* Criação de getters e setters
* Desenvolvimento de métodos para adicionar estoque e realizar vendas
* Validação de quantidade e controle de estoque
* Testes com cenários positivos, negativos e casos extremos

## ▶️ Como executar

1. Acesse a pasta do exercício desejado
2. Compile o arquivo:
   javac Main.java
3. Execute:
   java Main

## 📌 Observações

Os exercícios foram desenvolvidos durante a disciplina de Programação Orientada a Objetos, com foco na consolidação da lógica de programação em Java e na aplicação prática dos conceitos fundamentais da orientação a objetos.
